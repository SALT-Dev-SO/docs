## Spectrum Financial Forecast Model

SALT's financial projections are driven by the Spectrum FFM — an openpyxl-parsed Excel model that feeds directly into the D&O Dashboard analytics pages.

## Revenue Segments

The model tracks revenue across multiple lending segments:

| Segment | Description |
|---------|-------------|
| Interest Income | Net interest from crypto-backed loans |
| Origination Fees | Loan origination and processing fees |
| Collateral Management | Custody and collateral service fees |
| Exchange Revenue | Trading and conversion fees |
| API/Platform | Enterprise API access and platform fees |
