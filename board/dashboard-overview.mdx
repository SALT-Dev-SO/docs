## Dashboard Overview

The SALT D&O Dashboard is a Flask-based web application deployed on Railway that serves as the central governance platform for SALT Lending's board of directors and officers.

**Version:** 2.0 | **Platform:** Railway Production | **Auth:** Google OAuth + Password

## Navigation

The dashboard organizes features into these sidebar groups:

| Section | Pages | Description |
|---------|-------|-------------|
| Dashboard | Home | Main dashboard with treasury overview |
| Equity & Grants | Cap Table, Comparison, Options Grants, Scenarios | Equity management suite |
| FFM Analytics | Overview, Valuation, Redemption, Comp Table | Financial forecast pages |
| Presentations | Strategy Deck, BoD Jan 2026, Lightwave SPAC | Board presentations |
| Settings | Connected Spreadsheets | Data source configuration |

## Access Levels

| Role | Permissions |
|------|------------|
| **Admin** | Full access — all pages, API endpoints, content governance preview, grant CRUD |
| **Dashboard User** | Read-only — view pages, run calculations, export data |
| **Unauthenticated** | Redirect to login — no access |
